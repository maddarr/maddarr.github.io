Back in 2018 I began looking into programming in the Swift language, I did a few tutorials on line and finally found a guy that had a great library of both books and tutorials on the subject Hacking With Swift. I originally began by following one of his books, Hacking with iOS, and had a blast learning from it. However because of not wanting to publish any work to the apple app store and not being able to share the things I did with family or friends, I soon gave up... For a while. Roughly a year later I picked it back up again and began buying, reading, watching and learning how to program in Swift and Xcode once again, this time focusing on the macOS. and after a while I began to write my own games and code. Still not publising on the apple store, but able to run them from my own computer and show them off from there.

After playing around with Swift and Xcode, this last year I decided to take it further with the coding and I enrolled in an online training program called Codecademy. I began by learning the Python language and how to use the Command Line to help utilize the language.

Once I had finished the Python3 certification, I began to look into what other courses were available. I found one that would send me down the path I'm on now. Front End Engineer. Within this career path I have now achieved certifications in HTML, Basic and Intermediate CSS and Basic Java Script.
